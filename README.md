# Guia Rápido - Integração de APIs com CRM DataCrazy

Este projeto contém guias de integração passo a passo para conectar diferentes APIs de WhatsApp ao CRM DataCrazy. Cada guia é organizado em sua própria pasta, com um menu principal central para navegação.

## Propósito

Fornecer documentação clara e estruturada para integrar APIs de WhatsApp ao CRM DataCrazy, incluindo instruções detalhadas, capturas de tela e passos de validação.
